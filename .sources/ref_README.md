## Key Findings

1. Global green building market: $618.58B (2025) → $1.37T by 2034
2. LEED referenced in 71% of specs for projects >$50M
3. LEED buildings show 31% gross rent premium (4% net)
4. Construction cost premium declining to 0-2% at LEED Certified/Silver
5. Mid-size firms: 40-55% have some RE strategy (LOW confidence)
6. Electric construction equipment: $13.6-15.8B, 20-22% CAGR
7. All 6 case study firms started with LEED certification as foundation

## Confidence Levels

- **[HIGH]**: Multiple corroborating sources
- **[MEDIUM]**: Single reliable source or partial data
- **[LOW]**: Inference from related data or limited sourcing
